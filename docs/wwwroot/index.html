Introducing Adaptive Cards

Adaptive Cards is a JSON format which gives developers the ability deliver great lookin cards to any device or client.

Developer features

  • Simple by expressive JSON layout gives developers ability easily create rich great looking cards across any device or client!
  • Rich inut controls (text, date, time, etc) allows developer to collect information from users!
  • Speech enabled from Day one!
  • Open source libraries available via npm, nuget, etc. allows for easy integration into your application!
